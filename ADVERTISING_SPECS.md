# Технические требования для размещения рекламы

## 1. Баннеры

### Горизонтальный баннер (главная страница, верх)
- **Размер:** 1200x200 пикселей
- **Формат:** JPG, PNG, WebP
- **Максимальный вес:** 300 КБ
- **Позиция:** Под шапкой сайта, над основным контентом
- **Параметры:**
  ```javascript
  {
    id: string,              // Уникальный ID баннера
    title: string,           // Заголовок (опционально)
    imageUrl: string,        // URL изображения баннера
    linkUrl: string,         // URL ссылки при клике
    position: 'top' | 'middle' | 'bottom',
    startDate: Date,         // Дата начала показа
    endDate: Date,           // Дата окончания показа
    priority: number,        // Приоритет показа (1-10)
    targetBlank: boolean     // Открывать в новой вкладке
  }
  ```

### Вертикальный баннер (боковая панель)
- **Размер:** 300x600 пикселей
- **Формат:** JPG, PNG, WebP
- **Максимальный вес:** 200 КБ
- **Позиция:** Правая боковая панель каталога

### Квадратный баннер (между разделами)
- **Размер:** 600x600 пикселей
- **Формат:** JPG, PNG, WebP
- **Максимальный вес:** 250 КБ

---

## 2. Логотипы партнеров/рекламодателей

### Логотип в карточке партнера
- **Размер:** 120x120 пикселей (квадратный) или 200x80 пикселей (горизонтальный)
- **Формат:** PNG с прозрачным фоном, SVG
- **Максимальный вес:** 50 КБ
- **Параметры:**
  ```javascript
  {
    id: string,
    name: string,            // Название компании
    logo: string,            // URL логотипа или emoji
    description: string,     // Краткое описание компании
    specialization: string,  // Специализация
    website: string,         // Сайт компании
    advantages: string[],    // Преимущества (до 5)
    caseStudy: string,       // Кейс/статистика
    featured: boolean        // Показывать на главной
  }
  ```

### Логотип в слайдере партнеров
- **Размер:** 180x90 пикселей
- **Формат:** PNG с прозрачным фоном, SVG
- **Максимальный вес:** 30 КБ
- **Рекомендации:** Монохромный логотип с отступами для лучшего вида

---

## 3. Рекламные карточки предложений

### Карточка в каталоге
- **Изображение:** 400x200 пикселей (опционально)
- **Формат:** JPG, PNG, WebP
- **Максимальный вес:** 150 КБ
- **Параметры:**
  ```javascript
  {
    id: number,
    category: 'bank' | 'phone' | 'crm' | 'mobile' | 'other',
    partner: string,         // Название партнера
    partnerLogo: string,     // Emoji или URL логотипа
    title: string,           // Название предложения (до 60 символов)
    description: string,     // Описание (до 120 символов)
    price: string,           // Цена/условие
    oldPrice: string,        // Старая цена (для акций)
    features: string[],      // Ключевые фичи (до 5)
    rating: number,          // Рейтинг (1-5)
    reviews: number,         // Количество отзывов
    featured: boolean,       // Показывать на главной
    linkUrl: string,         // Партнерская ссылка
    badge?: string           // Бейдж "Акция", "Хит" и т.д.
  }
  ```

## 4. Нативная реклама (статьи/новости)

### Рекламная новость
- **Изображение:** 800x400 пикселей
- **Формат:** JPG, PNG, WebP
- **Максимальный вес:** 200 КБ
- **Параметры:**
  ```javascript
  {
    id: number,
    title: string,           // Заголовок (до 100 символов)
    date: string,            // Дата публикации
    badge: string,           // "Акция", "Партнер", "Спецпредложение"
    description: string,     // Краткое описание (до 200 символов)
    imageUrl?: string,       // URL изображения
    linkUrl: string,         // Ссылка на полную статью
    sponsored: boolean       // Флаг рекламного контента
  }
  ```

## 5. Общие требования

### Технические требования
- Все изображения должны быть оптимизированы для web
- Поддержка Retina-дисплеев (@2x версии)
- Адаптивность для мобильных устройств
- Формат WebP предпочтительнее для лучшей оптимизации

### Контентные требования
- Заголовки на русском языке
- Отсутствие грамматических ошибок
- Соответствие законодательству РФ о рекламе
- Пометка "Реклама" для коммерческого контента
